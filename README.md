## Other instructions

1. The script is developed and maintained in spare time. If you have any questions, you are welcome to give feedback, but no bad comments will be accepted.

2. If you find that the script is completely unavailable, it is generally not a problem with the script itself. It is recommended to delete it first and then install it again.
